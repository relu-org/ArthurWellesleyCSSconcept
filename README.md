###CSS

Slice on three files like:

| Structure           | Dimensions           | Visual         |
| ------------------- | -------------------- | ---------------|
| diaplay             | width                | color          |
| position            | height               | background     |
| float               | align                | border         |
| clear               | padding              | font           |
| flex                | margin               | text           |
|                     | line-height          | box-shadow     |
